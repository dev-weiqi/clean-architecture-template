# clean-architecture-template

A template for a new app project

## Architecture / Libs

- Developed with Kotlin
- Clean Architecture and Modularization
- Coroutines / Flow for asynchronous
- Dependency Injection with Dagger Hilt
- Android Jetpack
- Retrofit / OkHttp
- Moshi

## Test Framework

- MockK
- Turbine
